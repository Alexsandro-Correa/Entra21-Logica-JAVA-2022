# Aprendendo sobre else if

## Incluindo novas perguntas no else de cada if

> Não há limites em relação a aumentos de condições em ifs, porém deve ter cuidado com a quantidade
> pois fica a pergunta (não seria melhor um switch?)
>
> porém quando utilizamos else if()
> se faz necessário fechar essa cadeia com um else final, por questões de boas práticas


`````java
if(condicao1){
	//se atender a condicao1
}else if (condicao2){
	//se atender a condicao2
	}else if (condicao3) {
	//se atender a condicao3
	}else (condicao4){
	//se atender a condicao
	}
	````
