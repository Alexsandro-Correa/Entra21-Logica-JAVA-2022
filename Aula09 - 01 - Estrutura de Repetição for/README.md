## Aula 09 Estrutura de Repetição for

for(int contador = 1; contador <= 1000; contador++){

   //execute varias vezes aqui

}

> Ideal para repetições com base na quantidade
>
> 1. Variável com ponto de início
> 2. Condição de permanência
> 3. Incremento ou decremento, pensando na condição da permanência
